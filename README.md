## Getting Started

### Install

```bash
$ npm install
```

### Build

```bash
$ npm run build
```

### Start

```bash
$ npm start
```

### Builds the app for production to the docker container

```bash
$ npm run docker-build
```

### Run container built from "docker-build"

```bash
$ npm run docker-run
```
